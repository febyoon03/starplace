# StarPlace — 사주·자미두수 기반 나에게 맞는 지역 추천

> **학번:** 2024510007 | **이름:** 이윤경 | **광운대학교 정보융합학과**

## 프로젝트 소개
생년월일시 기반 사주팔자 + 자미두수 명궁 + 성향 테스트 10문항을 종합해
전 세계 80개 도시 중 나에게 가장 잘 맞는 곳을 찾아주는 Streamlit 웹 앱입니다.

## 실행 방법
pip install -r requirements.txt
streamlit run app.py

## 필수 기능
- 로그인: SQLite 기반 아이디/비밀번호
- 캐싱: @st.cache_data 적용 (사주, 자미두수, 지역DB)
- 퀴즈: 성향 테스트 10문항 → 오행 벡터 → TOP5 지역 추천
